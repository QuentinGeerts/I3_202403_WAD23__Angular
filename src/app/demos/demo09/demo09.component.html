Démonstration 09 - Les directives structurelles

Il existe 3 structures :

  • *ngIf : affiche un élément en fonction d'une expression booléenne
  • *ngFor : permet de répéter un élément un certains nombres de fois
  • *ngSwitch : permet d'afficher plusieurs éléments HTML en fonctione d'une valeur
