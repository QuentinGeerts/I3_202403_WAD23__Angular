Démonstration 01 - Interpolation de chaines (One-Way binding)

L'interpolation, notée avec des doubles accolades {{ expression }}, permet d'insérer dynamiquement des valeurs variables dans le contenu HTML.

Récupération des valeurs se fait de TypeScript vers HTML

Affichage des valeurs :

Type primitif : string {{ variable1 }}

Type primitif : number {{ variable2 }}

Type primitif : boolean {{ variable3 }}

Array : string[] {{ variable4 }}

Array : number[] {{ variable5 }}

Array : boolean[] {{ variable6 }}

Definite assigment assertion : string {{ variable7 }}

Ambient declaration : number {{ variable8 }}

Any : any[] {{ variable9[0] }}

Any : any[] {{ variable9[1] }}

Any : any[] {{ variable9[2]() }}

Méthodes : function anonyme {{ variable10() }}

Méthodes : function fléchée {{ variable11() }}

Méthodes : a: 5 et b: 3 {{ variable12(5, 3) }}

Méthodes : a: 5 et b: 3 {{ variable13(5, 3) }}
