Démonstration 06 - Les pipes

Les pipes sont des fonctions qui permettent de transformer une expression dans la vue.

Pour utiliser une pipe, vous devez écrire {{ expression | nomPipe }}

Exemples :

Pipe string

Uppercase : {{ "Bonjour tout le monde. Je m'appelle Quentin." | uppercase }}

Lowercase : {{ "Bonjour tout le monde. Je m'appelle Quentin." | lowercase }}

Titlecase : {{ "Bonjour tout le monde. Je m'appelle Quentin." | titlecase }}

Pipe Number | Percent

{{ expression | number : digitsInfo : locale }}

digitsInfo : minIntegerDigits . minFractionDigits - maxFractionDigits

Avant : {{ 1234.5678 }}

Après : {{ 1234.5678 | number }}

Après : {{ 1234.5678 | number : '5.1-2' }}

Après : {{ 1234.5678 | number : '5.1-2' : 'fr-FR' }}

Après : {{ 1234.5678 | percent : '5.1-2' : 'fr-FR' }}

Pipe Date

Avant : {{ today }}

Après : {{ today | date : 'dd MMMM yyyy' }}

Après : {{ today | date : 'dd MMMM yyyy HH:mm:ss' : 'GMT+1' }}

Après : {{ today | date : 'dd MMMM yyyy HH:mm:ss' : 'GMT+8' }}

Après : {{ today | date : 'dd MMMM yyyy HH:mm:ss' : 'GMT-8' }}

Après : {{ today | date : 'dd MMMM yyyy HH:mm:ss' : 'GMT+1' : 'fr-FR' }}

Après : {{ today | date : 'dd MMMM yyyy HH:mm:ss' : 'GMT+1' : 'es-ES' }}

Pipe Currency

Avant : {{ 1234.56 }}

Après : {{ 1234.56 | currency }}

Après : {{ 1234.56 | currency : 'EUR' }}

Après : {{ 1234.56 | currency : 'USD' }}

Après : {{ 1234.56 | currency : 'JPY' }}

Après : {{ 1234.56 | currency : 'EUR' : 'symbol' }}

Après : {{ 1234.56 | currency : 'EUR' : 'code' }}

Après : {{ 1234.56 | currency : 'EUR' : 'symbol-narrow' }}

Après : {{ 1234.56 | currency : 'EUR' : 'symbol' : '1.1-3' }}

Après : {{ 1234 | currency : 'EUR' : 'symbol' : '1.1-3' }}

Après : {{ 1234.5665 | currency : 'EUR' : 'symbol' : '1.1-3' }}

Après : {{ 1234.56 | currency : 'EUR' : 'symbol' : '1.1-3' : 'fr-FR' }}

Pipe JSON

Avant : {{ quentin }}

Après : {{ quentin | json }}
